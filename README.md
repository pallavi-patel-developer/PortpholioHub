## ⚝ About the Project

**PortfolioHub** is a comprehensive platform designed for both Job Seekers (interns, freelancers, professionals) and Employers. It provides a seamless experience for building stunning portfolios, discovering top talent, and streamlining the hiring process.

Whether you're looking to apply for an internship, showcase your projects, or find the perfect candidate for your company, PortfolioHub has you covered with its intuitive dashboards and robust backend.


---

## ⚝ Key Features

- **Dual User Roles**: Dedicated experiences for Job Seekers and Employers.
- **Portfolio Builder**: Easily create and customize your professional portfolio.
- **Candidate Directory**: Browse through a rich directory of talent with filtering options (Location, Industry, Experience, Skills).
- **Internships & Jobs**: Post opportunities or apply for them with interactive workflows.
- **Secure Authentication**: Traditional Email/Password (JWT) + Social Logins (Google & GitHub) powered by Passport.js.
- **Rich Dashboards**: Manage everything—applications, interviews, and candidate tracking—in one place.
- **Media Uploads**: Secure profile and portfolio image uploads using Cloudinary.

---

## ⚝ Tech Stack

### Frontend
- **HTML5 & CSS3**: Responsive, mobile-first design.
- **Vanilla JavaScript**: Core logic and DOM manipulation.
- **GSAP**: Smooth, high-performance web animations.

### Backend
- **Node.js & Express.js**: Fast and scalable server architecture.
- **MongoDB & Mongoose**: NoSQL database for flexible data modeling.
- **Passport.js**: Robust authentication strategies (Local, JWT, OAuth).
- **Cloudinary**: Cloud-based image management.
- **Nodemailer**: Email notifications and verification.

## ⚝ Usage

1. **Sign Up**: Create an account as either an intern/professional or an employer.
2. **Build Profile**: Navigate to the "Build Portfolio" section to add your skills, experiences, and projects.
3. **Explore**:
   - *Employers*: Browse the "Talents" directory to find suitable candidates.
   - *Candidates*: Browse the "Internships" section to find and apply for roles.
4. **Dashboard**: Use your personalized dashboard to track applications, manage job postings, or update your portfolio.
